═══════════════════════════════════════════════════════════════════

WHAT IS THIS FILE:
------------------

postcss.config.js tells Vite to:
1. Process CSS with Tailwind CSS
2. Add vendor prefixes with Autoprefixer

Without this file:
❌ Tailwind CSS classes are NOT compiled
❌ Your @import "tailwindcss" does NOTHING
❌ Website shows plain HTML

With this file:
✅ Tailwind CSS processes all your styles
✅ All classes get compiled
✅ Beautiful styled website

═══════════════════════════════════════════════════════════════════

FILE CONTENT:
-------------

export default {
  plugins: {
    tailwindcss: {},
    autoprefixer: {},
  },
}

That's it! Just 5 lines!
