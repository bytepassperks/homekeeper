YOUR ERROR:
-----------
"Rollup failed to resolve import "sonner@2.0.3""

THE CAUSE:
----------
Your files have @version in imports (Figma Make syntax)
Regular deployment can't handle this

THE FIX (30 SECONDS):
---------------------

1. Open VS Code in your homekeeper folder

2. Press: Ctrl+Shift+H

3. Click the .* button (enables Regex)

4. Type in Find box:     @\d+\.\d+\.\d+

5. Leave Replace box:    EMPTY

6. Type in files box:    *.tsx,*.ts

7. Click: "Replace All" button

8. Run these commands:
   git add .
   git commit -m "Fix: Remove versioned imports"
   git push origin main

9. Wait 5 minutes

10. YOUR APP IS LIVE! 🎉

---------------

Removes all @1.2.3 style versions from:

                               ^^^^^^^^ removes this

Becomes:
