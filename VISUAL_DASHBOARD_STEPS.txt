
🍼 BABY STEPS: Deploy to Supabase Dashboard
═══════════════════════════════════════════════════════════════


STEP 1: Open Supabase
┌─────────────────────────────────┐
│  https://supabase.com/dashboard │
│                                 │
│  → Login                        │
│  → Click "HomeKeeper" project   │
└─────────────────────────────────┘
                 ↓


STEP 2: Go to Edge Functions
┌─────────────────────────────────┐
│  Left Sidebar                   │
│  → Click "Edge Functions"       │
└─────────────────────────────────┘
                 ↓


STEP 3: Create New Function
┌─────────────────────────────────┐
│  → Click "Create a new function"│
│  → Function name: server        │
│  → Click "Create"               │
└─────────────────────────────────┘
                 ↓


STEP 4: Paste index.ts
┌─────────────────────────────────┐
│  Code Editor appears            │
│  → Delete all existing code     │
│  → Open: supabase/functions/    │
│           server/index.ts       │
│  → Select All (Ctrl+A)          │
│  → Copy (Ctrl+C)                │
│  → Paste in Supabase (Ctrl+V)  │
└─────────────────────────────────┘
                 ↓


STEP 5: Add kv_store.ts
┌─────────────────────────────────┐
│  → Click "+ Add file" button    │
│  → Filename: kv_store.ts        │
│  → Open: supabase/functions/    │
│           server/kv_store.ts    │
│  → Select All (Ctrl+A)          │
│  → Copy (Ctrl+C)                │
│  → Paste in Supabase (Ctrl+V)  │
└─────────────────────────────────┘
                 ↓


STEP 6: Deploy
┌─────────────────────────────────┐
│  → Click "Deploy" button        │
│     (green button, top-right)   │
│  → Wait 30-60 seconds           │
│  → See success message ✓        │
└─────────────────────────────────┘
                 ↓
